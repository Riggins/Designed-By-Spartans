Designed by Spartans is a project to create a design community for SJSU design students. This is an open-source WordPress theme.

---

User Types:
* DSGD Students - Want to see what others are doing, get help/clarification, inspiration, jobs, community
* Outsiders - Alums, unrelated, who want to see what the program is doing, interact with the community

Goals of the site:
* Community - feedback (comments, +), events, jobs
* Inspiration - see what others are doing
* Collaborate - find out projects that are looking for help
